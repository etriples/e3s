# e3s

## Required Packages
* libpng-dev
* libpng++-dev
* make
* g++

```bash
$ sudo apt-get install libpng-dev libpng++-dev make g++
```

## Installation & Running
```bash

$ git clone https://github.com/etriples/e3s
$ cd e3s
$ make
$ make run
```
`This repository is compiled in Ubuntu 22.04, please recompile using make`
